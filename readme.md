## フォントファイルから使いたい文字を抽出する

#### 用途
フォントファイルが重いので使用したい文字のみを抽出し使用する    

#### 注意事項
ファイル名はそれぞれ変更すること    
著作権などはフォント製作元、製作者、に配慮すること      
フォントファイルの置き場所はsrc配下     

### 使い方
python main.py で実行以下コマンドを指定する

- show ファイル名.ttf   
 ファイル名を指定して使用できる文字の一覧を取得     
 txtファイルで出力

- extraction 抽出元ファイル名.ttf 抽出したい文字.txt 出力ファイル名.ttf    
 それぞれのファイル名を指定して抽出元ファイルから抽出したい文字を切り取って出力ファイルに出力する
